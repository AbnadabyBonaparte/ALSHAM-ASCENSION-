# I. MANIFESTO DA EXPERIÊNCIA

A ALSHAM não projeta páginas. Projeta **estados emocionais**.

O visitante não deve sentir que está sendo vendido algo. Deve sentir que está sendo **avaliado para pertencer** a algo maior — e que o conhecimento que carrega pode, pela primeira vez, ser tratado como patrimônio.

---

# II. MAPA EMOCIONAL POR MOMENTO

## Ao entrar (0–3 segundos)

| Dimensão | Especificação |
|----------|---------------|
| **Emoção alvo** | Solenidade, antecipação, gravidade calma |
| **Emoção proibida** | Excitação, urgência, confusão, casualidade |
| **Sensação física** | "Entrei em um lugar importante" |
| **Ritmo** | Loader 1.5s — ritual, não espera frustrante |
| **Visual** | Escuro, ouro sutil, silêncio visual |
| **Copy** | Nenhuma — apenas símbolo e filete |

## Ao ler o Hero (3–15 segundos)

| Dimensão | Especificação |
|----------|---------------|
| **Emoção alvo** | Identificação profunda, aspiração silenciosa |
| **Emoção proibida** | FOMO, hype, "preciso comprar agora" |
| **Pergunta interna do visitante** | "Isso é sobre mim — sobre o que eu realmente construí?" |
| **Ritmo** | Pausado — headline em 2 linhas, respiração entre blocos |
| **CTA emocional** | Candidatura, não compra |

## Ao navegar (scroll)

| Dimensão | Especificação |
|----------|---------------|
| **Emoção alvo** | Clareza crescente, confiança processual |
| **Emoção proibida** | Sobrecarga, fadiga visual, skepticism por excesso |
| **Ritmo** | Ondulante — denso (paradoxo) → pausa (tese) → estruturado (jornada) |
| **Transições** | Golden Filament como respiração entre seções |
| **Profundidade** | Cada seção = mudança de ambiente/arquitetura |

## Ao compreender a metodologia

| Dimensão | Especificação |
|----------|---------------|
| **Emoção alvo** | Segurança, "existe um caminho", respeito pelo processo |
| **Emoção proibida** | "É complexo demais", "é mais um funil" |
| **Sensação** | Engenharia confiável — como blueprint de edifício |

## Ao chegar ao CTA (ALSHAM SELECT™)

| Dimensão | Especificação |
|----------|---------------|
| **Emoção alvo** | Exclusividade merecida, compromisso solene |
| **Emoção proibida** | Pressão, desespero comercial, formulário genérico |
| **Sensação** | "Estou me candidatando a um instituto, não preenchendo lead form" |
| **Visual** | Câmara dos Ativos — cofre, selo, filete |
| **Microcopy pós-envio** | "Candidatura recebida" — não "Obrigado, entraremos em contato!" |

## Ao concluir / sair

| Dimensão | Especificação |
|----------|---------------|
| **Emoção que permanece** | Respeito, curiosidade intelectual, aspiração de legado |
| **Emoção que permanece** | "Essa empresa é diferente de tudo que vi" |
| **Emoção proibida residual** | Arrependimento, pressão, sensação de ter sido manipulado |
| **Memorável** | Uma imagem (observatório, cofre, filete dourado) + uma frase |

# VI. PRINCÍPIOS DE INTERAÇÃO

| Princípio | Especificação |
|-----------|---------------|
| **Ritual** | Loader, filete draw, counter — movimentos com significado |
| **Resposta** | Hover: elevação 4px — objeto ganha presença, não "pula" |
| **Feedback** | Formulário: confirmação solene, não confete |
| **Velocidade** | Animações 600ms — nunca < 200ms (ansiedade) nem > 1000ms (frustração) |
| **Som** | Opcional, muted default — room tone de biblioteca se ativado |
| **Cursor** | Desktop: círculo dourado 24px — presença, não gimmick |

---

# VII. ACESSIBILIDADE EMOCIONAL

- Exclusividade **não pode** excluir por barreira técnica (WCAG AA)
- Solenidade **não pode** ser confundida com frieza hostil
- Curadoria **deve** explicar por que o processo existe (tese, paradoxo)
- Candidatura **deve** ser alcançável em ≤ 6 campos, sem fricção desnecessária

# VIII. MÉTRICAS DE EXPERIÊNCIA

| Métrica | Indicador de sucesso |
|---------|---------------------|
| Tempo no hero | > 8s (leitura, não bounce) |
| Scroll depth | > 70% (narrativa funcionando) |
| CTA click rate | Qualificado > quantidade |
| Form completion | > 60% dos que iniciam |
| Return visit | Sinal de memorabilidade |
| Qualitative | "Parece uma holding, não uma agência" |
