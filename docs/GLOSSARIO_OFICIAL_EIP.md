# TERMOS (15)

---

## T1 — EIP

| Campo | Conteúdo |
|-------|----------|
| **Definição** | Framework metodológico e epistemológico que define como conhecimento humano é identificado, estruturado, validado, operado, mensurado e evoluído como patrimônio. Modelo de Conhecimento supremo do ecossistema. |
| **Contexto** | Camada acima da ALSHAM. Evolui por Evidências (VER). Sigla: usar **EIP** ou **Engineering of Intellectual Patrimony** (contexto internacional interno) até ADR-0001 — sem outra expansão congelada. |
| **Exemplos** | Ciclo Hipótese→Evidência→Canon; maturidade de conceitos; KGR |
| **Não-exemplos** | ALSHAM (organização); CRM; landing page; marca comercial primária |
| **Relações** | → ALSHAM (aplica); ← Evidências (retroalimenta); contém Glossário, Canon |
| **Documentos** | MAD-1.0 §III.1; ADR-0001 |
| **Histórico** | v1.0 MAD-1.0; expansão sigla ABERTA |

---

## T2 — Conhecimento

| Campo | Conteúdo |
|-------|----------|
| **Definição** | Totalidade do saber, experiência, história, competência e credibilidade de um sujeito — no estado **disperso, invisível e subutilizado** antes da engenharia. |
| **Contexto** | Estágio inicial da jornada. Matéria-prima, não produto final. |
| **Exemplos** | Expertise de 20 anos; metodologia não documentada; reputação fragmentada |
| **Não-exemplos** | Seguidores; views; post viral; diploma isolado sem estruturação |
| **Relações** | → Autoridade (quando estruturado); material do Projeto de Engenharia |
| **Documentos** | DOC-01; MAD IX |
| **Histórico** | v1.0 consolidado do dossiê institucional |

---

## T3 — Capital Intelectual

| Campo | Conteúdo |
|-------|----------|
| **Definição** | Conhecimento **estruturado como valor estratégico e econômico** — intermediário mensurável entre autoridade reconhecida e patrimônio formalizado. |
| **Contexto** | Termo de ponte na cadeia de valor intelectual. |
| **Exemplos** | Metodologia proprietária nomeada; audiência qualificada; marca profissional consolidada |
| **Não-exemplos** | Capital financeiro; valuation de startup; followers |
| **Relações** | Conhecimento → Autoridade → **Capital Intelectual** → Patrimônio |
| **Documentos** | DOC-02; MAD II |
| **Histórico** | v1.0 |

---

## T4 — Ativo Intelectual

| Campo | Conteúdo |
|-------|----------|
| **Definição** | Unidade **operável e mensurável** de capital intelectual — passível de gestão, licenciamento, proteção e escala. |
| **Contexto** | Após engenharia; antes era apenas potencial. |
| **Exemplos** | Curso licenciável; método registrado; agente cognitivo treinado no DNA™ |
| **Não-exemplos** | Ideia não documentada; rascunho sem estrutura; IP não protegida por negligência |
| **Relações** | Componente de Patrimônio Intelectual; output de GENESIS™, LEGACY™ |
| **Documentos** | MAD III.6; dossiê |
| **Histórico** | v1.0 |

---

## T5 — Patrimônio Intelectual

| Campo | Conteúdo |
|-------|----------|
| **Definição** | Sistema **integrado, protegido e perene** de ativos intelectuais — incluindo PI formal, Vault™, marcas, metodologias e receita recorrente desacoplada do tempo. |
| **Contexto** | Objetivo da transformação ALSHAM. Estágio antes do Legado. |
| **Exemplos** | Knowledge Vault™; portfólio de cursos + livros + certificações; licenciamentos CAPITAL™ |
| **Não-exemplos** | Perfil Instagram; site sem estratégia; renda apenas por hora |
| **Relações** | ← Conhecimento (origem); → Legado; Câmara dos Ativos (visual) |
| **Documentos** | DOC-01; DOC-03B (semântica símbolo) |
| **Histórico** | v1.0 |

---

## T6 — Engenharia

| Campo | Conteúdo |
|-------|----------|
| **Definição** | Aplicação **sistemática, intencional e baseada em evidências** de método sobre conhecimento — oposto a achismo, improviso e viralização. |
| **Contexto** | Termo nuclear: Engenharia de Autoridade, EIP, Projeto de Engenharia. |
| **Exemplos** | 9 etapas ALSHAM; ALSHAM DNA™; curadoria SELECT |
| **Não-exemplos** | "Hacks"; truques de growth; postagem aleatória |
| **Relações** | Define operação ALSHAM; princípio P8 MAD |
| **Documentos** | DOC-01; differentiation thesis |
| **Histórico** | v1.0 |

## T7 — ALSHAM

| Campo | Conteúdo |
|-------|----------|

| **Contexto** | Marca e operação. Subordinada ao EIP. |
| **Exemplos** | GENESIS™, SELECT™, holding comercial |
| **Não-exemplos** | Agência de marketing; EIP (modelo); freelancer |

## T13 — Projeto de Engenharia

| Campo | Conteúdo |
|-------|----------|
| **Definição** | **Instância aplicada** do EIP/ALSHAM para um especialista ou organização — com escopo, etapas, entregas e evidências. |

| **Exemplos** | Cliente médico em GENESIS→SENTINEL; implementação Vault |
